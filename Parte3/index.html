El lenguaje del futuro

La programación está detrás de todos los productos digitales que usamos día a día. Nuetras coders aprenden HTML5, CSS3, Javascript y el uso de manejadores de contenido, terminando preparadas para crear infinitos productos y soluciones web.

Emprendimiento

El código y el emprendiemiento van de la mano. Nuestras coders se convierten en emprendedoras digitales desde el primer día. El proyecto final de Laboratoría consiste en sacar adelante un Producto mínimo viable al mercado.

Aprender a parender

Para triunfar como desarrolladoras web nuestras coders se convierten en autodidactas, descubriendo que no hay limites para seguir creciendo en este campo. Una camputadora, conexión internet, grandes sueños y dedicación es todo lo que se necesita.
